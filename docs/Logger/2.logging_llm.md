# Logging LLM Interractions

Setup your api key in a dot env file with a name like DATAIDEA_API_KEY

Read the api key

```py
import os 

api_key = os.getenv('DATAIDEA_API_KEY')
```

## Log an interraction

```py
llm_log({
        'api_key': api_key,
        'user_id': '1234567890',
        'source': 'gpt-4o',
        'query': 'This is a test query',
        'response': 'This is a test response',
    })
```
