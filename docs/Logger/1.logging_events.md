# Logging Events 

Setup your api key in a dot env file with a name like DATAIDEA_API_KEY

Read the api key

```py
import os 

api_key = os.getenv('DATAIDEA_API_KEY')
```

## Log an event

```py
login('jumashafara', 'Chappie@256')
    
    event_log({
        'api_key': api_key,
        'user_id': '1234567890',
        'message': 'This is a test message',
        'level': 'info',
        'metadata': {'test': 'test'}
    })
```
